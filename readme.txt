Multi Payment Gateway connector for WooCommerce to accept payments.

== Description ==

Your first and last Payment Gateway to accept payments through multiple payment provider like Stripe, Gocardless, Coinbasecommerce, and many more.

== Requirements ==

This plugin requires a multi-payment-gateway installation.
